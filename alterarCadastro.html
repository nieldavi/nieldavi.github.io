Alterar cadastro

Atores:

Cliente

Pré-condição:

O cliente deve ter um cadastro no site.

Fluxo de eventos principal:

  1. O caso de uso começa quando o usuário clica no botão "Meus dados".
  2. O cliente clica no botão "Alterar dados".
  3. O sistema exibe os dados do cliente.
  4. O cliente altera os dados desejáveis.

Pós-condição:

Os dados do cliente foram alterados.

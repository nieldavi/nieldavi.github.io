AgroMercado

Esse sistema se propõe a realizar uma aplicação web para as associações de agricultura familiar, visando atender as necessidades das comunidades, como também expandir seus negócios, aumentar a visibilidade e facilitar o relacionamento entre clientes e os produtores.

Casos de uso:

    • Cadastrar Fornecedor
    • Excluir cadastro
    • Alterar cadastro
    • Visualizar cadastro
    • Cadastrar produto
    • Excluir produto
    • Alterar produto
    • Visualizar produto
    • Inserir fotos
    • Excluir fotos
    • Visualizar contatos
    • Visualizar galeria

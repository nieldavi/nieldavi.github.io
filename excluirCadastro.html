Excluir cadastro

Estende

5. Visualizar cadastro, passo 3

Pré-condição:

O cliente deve ter um cadastro no site.

Fluxo de eventos principal:

  1. O caso de uso começa quando o usuário clica no botão "Meus dados".
  2. O cliente clica no botão "Excluir conta".
  3. O sistema exibe uma caixa de mensagem com a confirmação de exclusão da conta.
  4. Com a confirmação do cliente a conta é excluída.

Pós-condição:

O cliente não possui mais conta no sistema.

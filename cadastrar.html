Pré-condição:

Fluxo de eventos principal:

  1. O caso de uso começa quando o usuário clica no botão "cadastre-se".
  2. O cliente preenche o formulário de cadastro do site.
  3. O sistema verifica se todos os campos foram digitados corretamente.
  4. O cliente clica no botão "cadastrar" e seu cadastro é gerado.
  5. O sistema exibe uma caixa de mensagem com a confirmação do cadastro.

Cadastrar Produto

Atores:

Administradores

Pré-condição:

Ser administrador do sistema

Fluxo de eventos principal:

  1. O caso de uso começa quando o administrador clica no botão "Produtos".
  2. Em seguinda é exibido na tela os produtos que estão cadastrados no sistema.
  3. O administrador clica no botão "Cadastrar produto".
  4. O administrador preenche o formulário com as especificações do produto.
  5. O administrador clica no botão "Adicionar".
  6. O sistema verifica se todos os campos foram digitados corretamente.
  7. O sistema exibe uma caixa de mensagem com a confirmação do cadastro de produto.

Pós-condição:

O produto é cadastrado no sistema.

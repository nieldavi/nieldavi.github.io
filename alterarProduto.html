Alterar Produto

Atores:

Administradores

Pré-condição:

Ser administrador do sistema

Fluxo de eventos principal:

  1. O caso de uso começa quando o administrador clica no botão "Produtos".
  2. Em seguinda é exibido na tela os produtos que estão cadastrados no sistema.
  3. O administrador escolhe o produto que deseja alterar.
  4. O administrador seleciona o produto e clica no botão "Alterar produtos".
  5. O administrador irá fazer as alterações desejadas.
  6. O administrador clica no botão "adicionar alteração".
  7. O sistema exibe uma caixa de mensagem com a confirmação de alteração do produto.

Pós-condição:

O produto é alterado no sistema.

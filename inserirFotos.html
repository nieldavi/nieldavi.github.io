Inserir Fotos

Atores:

Administradores

Pré-condição:

Ser administrador do sistema

Fluxo de eventos principal:

  1. O caso de uso começa quando o administrador clica no botão "Galeria".
  2. Em seguinda é exibido na tela todas fotos que estão cadastrados no sistema.
  3. O administrador clica no botão "Inserir Fotos".
  4. O administrador preenche o formulário com as especificações da foto.
  5. O administrador clica no botão "Adicionar".
  6. O sistema verifica se todos os campos foram digitados corretamente.
  7. O sistema exibe uma caixa de mensagem com a confirmação do inserção de foto.

Pós-condição:

A foto é inserido no sistema.
